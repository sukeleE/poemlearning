# 诗境长卷

一个融合古典诗词学习、3D 交互、创作练习与 AI 辅助的网页项目。  
前端为静态页面，后端提供 SQLite 数据服务与 AI 接口代理。

## 功能简介

- 古诗内容浏览与分章节学习（将进酒、登高、山居秋暝、宿建德江等）
- 3D 场景交互（古亭、鼓楼、布告板）
- 账号注册/登录、文章发布、点赞、收藏、评论
- AI 对话与创作辅助（后端转发 DeepSeek 接口）

## 目录结构

- `index.html`：前端入口
- `assets/`：样式、脚本、图片、音频等静态资源
- `backend/server.js`：后端 API 服务
- `data/app.sqlite`：SQLite 数据文件
- `Dockerfile`：容器化部署配置

## 环境要求

- Node.js 22+
- npm 10+（通常随 Node 安装）

## 本地安装与启动

在项目根目录执行：

```bash
npm install
npm start
```

默认后端地址：

- `http://127.0.0.1:3000`
- 健康检查：`http://127.0.0.1:3000/api/health`

## 前端访问

本项目前端是静态页面：

- 直接打开 `index.html` 即可进入页面
- 或使用任意静态文件服务打开该页面

前端默认请求 `http://127.0.0.1:3000/api`，所以请确保后端已启动。

## AI 功能配置（DeepSeek）

后端 AI 接口依赖环境变量 `DEEPSEEK_API_KEY`。未配置时，AI 功能会返回报错。

### PowerShell（当前终端会话生效）

```powershell
$env:DEEPSEEK_API_KEY="你的_deepseek_key"
npm start
```

## Docker 启动（可选）

```bash
docker build -t poetry-scroll-app .
docker run --rm -p 3000:3000 -e DEEPSEEK_API_KEY=你的_key poetry-scroll-app
```

## 数据说明

- 后端使用 SQLite，数据默认保存在 `data/app.sqlite`
- 启动时自动建表
- 若存在旧的 `data/db.json`，会在数据库为空时自动迁移

## 网页操作指导

### 一、基础浏览

1. 打开页面后，向下滚动进入主内容区。
2. 使用顶部导航在不同章节间跳转（如“将进酒 / 登高 / 格律 / 意象 / 创作”）。
3. 部分章节可点击居中的竖排标题展开/收起详细内容。

### 二、诗词学习区操作

1. 在诗词页面中，点击具体诗句可查看对应解析卡片。
2. 点击作者名可打开作者卡片（生平、风格、代表作等）。
3. 子页面左下通常会出现“视频集”卡片，可切换并播放相关视频。

### 三、账号与文章功能

1. 点击右上角“登录”进入账户中心。
2. 可进行注册、登录、退出登录。
3. 进入 3D 场景后点击“布告板”，可查看文章列表、刷新列表、查看收藏。
4. 在文章相关区域可执行点赞、收藏、评论等操作。

### 四、创作功能

1. 在 3D 场景点击“鼓楼”打开“古诗创作”弹窗。
2. 选择诗体模板（五绝、七绝、律诗、词牌或自由创作）。
3. 填写题目并在创作区输入诗句（每句一行）。
4. 可执行“发表古诗 / 存为草稿 / 预览”。

### 五、AI 功能操作

1. 在创作区的 AI 助教面板中，按需填写 API Key（仅本地浏览器保存）。
2. 使用“意象提示 / 格律指导 / 整体优化 / 诗词评析”按钮获取建议。
3. 可将 AI 输出“替换”或“追加”到创作区。
4. 诗词赏析弹窗中可使用“生成深度赏析”获取课堂讲解稿。

### 六、3D 场景交互

- 鼠标悬停：模型高亮
- 鼠标点击：弹出对应功能窗口
- 鼠标移动：视差体验
- 滚轮：场景缩放

## 常见问题

- 页面能打开但数据功能不可用：后端未启动，请先 `npm start`
- AI 接口报错：未配置 `DEEPSEEK_API_KEY` 或 Key 无效
- 端口冲突：请修改 `PORT` 环境变量后再启动
